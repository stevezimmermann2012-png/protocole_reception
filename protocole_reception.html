Aide & Guide d'utilisation

Protocole de Réception SIA 118

📋 Comment remplir le formulaire

Le formulaire est divisé en 5 étapes. Remplissez chaque étape et appuyez sur "Suivant". Les champs marqués d'un * sont obligatoires.

🔢 Les 5 étapes

  1. Identification — Informations du projet, maître d'œuvre, entrepreneur
  2. Protocole de test — Date d'essai, résultat de la réception
  3. Instruction & Annexes — Personnes instruites, documents remis
  4. Signatures — Signez avec votre doigt ou la souris
  5. Récapitulatif — Vérifiez et téléchargez le PDF

📄 Générer le PDF

À l'étape 5, appuyez sur "Télécharger le PDF". Le PDF reproduit le modèle officiel SIA 118.

📲 Installer l'application

Android (Chrome) : Ouvrez le lien dans Chrome → bannière "Installer" → Appuyez sur Installer.


iPhone/iPad (Safari) : Ouvrez dans Safari → bouton partage ⬆ → "Sur l'écran d'accueil" → Ajouter.
